
# [TF Diagnose](https://nodiraibrogimova.github.io/TF-Diagnose/)
Online platform for a project with a target: AI systems + clinicians for an early-diagnosis
This platform includes the extended version of the design provided in [Figma](https://www.figma.com/file/0MW7v9ZfVjhqQ3f20kQbL2/Typhoid-Fever--early-diagnosis-%7C-Logo?node-id=0%3A1)

Built with Angular framework - a TypeScript-based free and open-source web application framework led by the Angular Team at Google and by a community of individuals and corporations.

The current version provides:

- [x] Design and the extended version of the design migrated to a website
- [x] SPA for an online platform foreseeing a template. Images have been shown in colored shapes. All are replaced once the assets are confirmed in the process of project development
- [x] Responsive layout: Desktop, tablet and mobile versions of the website
